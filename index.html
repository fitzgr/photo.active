Browser-based event booth foundation with an admin setup console and guest kiosk mode. This MVP runs entirely on your desktop with local storage and camera access.

Local Run

Serve the project on localhost so camera APIs work reliably:

py -m http.server 5500

Then open http://localhost:5500.
