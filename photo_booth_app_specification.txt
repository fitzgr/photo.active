
Event Photo Booth Web Application – Product Specification`r`n1. Project Overview`r`nA web‑based photo booth platform designed for event professionals. The system includes:`r`n• A secure admin backend for event configuration and management.`r`n• A public front‑end kiosk interface for guest interaction (photo booth experience).`r`n• Cloud galleries and media sharing.`r`n• Optional integrations including printers, AI features, and automated highlight videos.`r`nThe platform must be optimized for tablet use (especially iPad) but remain browser‑based.`r`n2. System Architecture`r`nFrontend (Guest Interface):`r`n• Browser‑based kiosk mode optimized for tablets.`r`n• Locked interface preventing navigation outside the event page.`r`n• Camera access for photo/video capture.`r`n• UI optimized for touch interaction.`r`nBackend (Admin Interface):`r`n• Secure login for administrators.`r`n• Event creation and configuration.`r`n• Media management and gallery controls.`r`n• Data export and analytics.`r`nServer / Processing Layer:`r`n• Media processing (GIF, boomerang, video compilation).`r`n• Storage for images, videos, galleries.`r`n• AI processing (future feature).`r`n• Scheduled automation (highlight reel generation).`r`n3. User Roles`r`nAdmin / Operator`r`n• Create and configure events.`r`n• Upload templates and overlays.`r`n• Configure galleries and sharing.`r`n• Export captured data.`r`nEvent Guest`r`n• Interact with kiosk interface.`r`n• Capture photos, GIFs, or videos.`r`n• Share media via SMS/email/QR.`r`n4. Event Management`r`nEvent Creation Fields:`r`n• Event name`r`n• Event date/time`r`n• Event end time`r`n• Event type (Wedding / Corporate / Private / Public)`r`n• Public or Private Gallery toggle`r`nEvent Controls:`r`n• “Go Live” toggle to activate kiosk interface.`r`n• Event‑specific gallery creation.`r`n• Custom event branding (logo, colors).`r`n• Event URL for gallery access.`r`n5. Photo & Video Capture Modes`r`nSupported Capture Types:`r`n• Single Photo`r`n• Multi‑photo Strip`r`n• GIF (4‑5 images auto animated)`r`n• Boomerang`r`n• Short Video (30 second testimonials)`r`n• Slow motion video (future)`r`n6. Template & Overlay System`r`nTemplate Upload:`r`n• Upload PNG/PDF overlays.`r`n• Position capture window(s).`r`n• Support for photo strips and multi‑photo layouts.`r`nOverlay Application:`r`n• Automatically applied to captured images.`r`n• Preview available during event setup.`r`n7. Screensaver / Idle Mode`r`nIdle Detection:`r`n• Detect inactivity via camera or time threshold.`r`nScreensaver Behavior:`r`n• Switch to animated overlay screen.`r`n• Live camera preview visible beneath overlay.`r`n• Call‑to‑action text displayed (e.g., “Come closer to start”).`r`nOverlay Library:`r`n• Built‑in animated overlays.`r`nExamples:`r`n• Fire`r`n• Sparkles`r`n• Abstract motion`r`n• Confetti`r`n• Brand‑neutral patterns`r`nAdmin can select overlay and customize text.`r`n8. Galleries`r`nGallery Types:`r`n• Public Gallery (e.g., weddings)`r`n• Private Gallery (corporate/internal)`r`nGallery Features:`r`n• Event branding (colors/logo).`r`n• Automatic upload of captured media.`r`n• Download and sharing options.`r`n• Admin moderation option (optional).`r`n9. Media Sharing`r`nSharing Methods:`r`n• SMS`r`n• Email`r`n• QR Code download`r`n• Social sharing (Instagram/Facebook manual share)`r`nGuest receives:`r`n• Their individual photo/video`r`n• Link to event gallery (if public)`r`n10. Data Capture`r`nOptional Data Collection before sharing:`r`n• Name`r`n• Email`r`n• Phone number`r`n• Company`r`n• Custom questions`r`nFeatures:`r`n• Admin can toggle fields on/off.`r`n• Required vs optional fields.`r`n• Consent checkbox for marketing use.`r`nExport:`r`n• CSV / Excel export.`r`n• Includes captured data and media filename.`r`n11. Automated Highlight Reel`r`nFeature Description:`r`nAutomatically generate a highlight video after event completion.`r`nConditions:`r`n• Only for Public Events.`r`nProcess:`r`n• Event ends.`r`n• Scheduled job runs next morning.`r`n• Selects images/videos from gallery.`r`n• Generates slideshow video.`r`nVideo Features:`r`n• Background royalty‑free music.`r`n• Branded intro/outro.`r`n• Dynamic transitions.`r`nDistribution:`r`n• Sent via email or SMS to guests who opted in.`r`n• Added to event gallery.`r`n12. Testimonial Video Compilation`r`nSeparate from highlight reel.`r`nProcess:`r`n• Collect all testimonial video clips.`r`n• Automatically compile into a single video.`r`nOutput:`r`n• Downloadable for event organizer.`r`n13. Printer Integration`r`nHardware Integration:`r`n• Bluetooth printer support.`r`n• Network printer support.`r`nFeatures:`r`n• Print immediately after capture.`r`n• Print layout matches selected template.`r`n• Multiple copies option.`r`nSystem should support common photo booth printers where possible.`r`n14. AI Feature (Future Phase)`r`nAI Background Generation:`r`n• Replace or generate background using AI.`r`n• Guests choose theme background.`r`nExample themes:`r`n• Beach`r`n• Space`r`n• Red carpet`r`n• Corporate branded scene`r`nImplementation:`r`n• AI image generation or background replacement service.`r`n15. Security`r`nAdmin access protected by login.`r`nBackend and kiosk interfaces separated.`r`nKiosk mode:`r`n• Prevent navigation to backend.`r`n• Locked browser session.`r`nData protection:`r`n• Secure storage for personal data.`r`n• Consent tracking for marketing usage.`r`n16. Device Support`r`nPrimary:`r`n• iPad Safari`r`n• Chrome tablet browsers`r`nSecondary:`r`n• Desktop browsers for admin panel.`r`n17. Media Storage`r`nStore:`r`n• Photos`r`n• GIFs`r`n• Videos`r`n• Highlight reels`r`nStructure:`r`n/events/{event_id}/photos`r`n/events/{event_id}/videos`r`n/events/{event_id}/gallery`r`n18. Future Enhancements`r`n• AI themed photo transformations`r`n• AR props and filters`r`n• Multi‑booth synchronization`r`n• Live slideshow display on large screens`r`n• CRM integrations`r`n• Analytics dashboard`r`n
